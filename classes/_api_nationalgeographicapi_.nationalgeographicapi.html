Static getLatestNews

  • getLatestNews(pageSize?: number, pageNumber?: number, cb?: Function): Promise<NewsPayload[]> | void
    • Defined in api/NationalGeographicAPI.ts:25

    Function that returns the latest news from National Geographic. Can retrieve articles in 'pages', not only the the very latest articles given combination of pageSize and pageNumber. If no pageSize or pageNumber is provided, it returns the latest 3 news articles

    Supports both Promise and callback based returning of result. If no callback is provided, it returns Promise with result.

    Parameters

    • Optional pageSize: number

      The number of articles to be returned from query. Defaults to 3.

    • Optional pageNumber: number

      The page number to retrieve, works in conjunction with pageSize to determine data.

    • Optional cb: Function

      Optional callback otherwise Promise based return.

    Returns Promise<NewsPayload[]> | void

Static getPhotoOfDay

  • getPhotoOfDay(day?: number | string | Function, cb?: Function): Promise<PhotoPayload | void>
    • Defined in api/NationalGeographicAPI.ts:48

    Function that returns the latest Photo of the Day from National Geographic. Can retrieve today's or any day in the past photo up to January 1st 2009 when Photo of the Day begins. Date can be in millis or UTC string.

    If there is no photo posted for the day yet due to timezone differences, it will retrieve the previous days photo until today's one is posted.

    Supports both Promise and callback based returning of result. If no callback is provided, it returns Promise with result.

    Parameters

    • Optional day: number | string | Function

      The historical day to retrieve data for. Defaults to retrieving today's photo.

    • Optional cb: Function

      Optional callback otherwise Promise based return.

    Returns Promise<PhotoPayload | void>
